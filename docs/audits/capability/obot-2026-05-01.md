## Section 1 — Tool identification

- **Name:** obot (agent runtime + orchestrator)
- **Container:** `obot`, image `obot/obot:latest`.
- **Compose location:** in-repo at `docker/obot-stack.yml`.

    is connected end-to-end.
  - Live traffic: shim → server requests observed (POST /mcp +
    DELETE /mcp from `.19.0.9`, the shim's IP).
  - 42 PIDs in the obot container — multi-process, runtime is alive
    and serving.
- **Stack-coverage hypothesis (from D-17-01):** "primary agent runtime"
  — **CONFIRMED.**
