## Two-Key Model (Worked Example #6)

The original single-master comparison frame was incorrect for Sonarr/Radarr.

- Sonarr/Radarr: Prowlarr-proxied consumer rows validate through a Prowlarr-managed application key path (`07ab59f4731b` in this run).
- Sportarr: rows currently validate against the Prowlarr master-key path (`a3051e37707a`).
- Therefore, hash mismatch versus Prowlarr master is not sufficient to declare drift.

Remediation trigger is now functional auth failure (`HTTP 401`) on consumer release probe, not hash mismatch.
