## Description

Add `dir='rtl'` detection based on the selected language and flip flex-direction and text-align properties via a `.rtl` class on the root element, ensuring Arabic/Hebrew layouts render correctly.

## Key requirements

- `RTL_LANGS = new Set(['ar', 'he', 'fa', 'ur'])` check in language-switch logic
- `document.documentElement.classList.toggle('rtl', isRtl)`
- CSS `.rtl` overrides for flex containers and kanban
