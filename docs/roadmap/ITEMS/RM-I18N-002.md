## Description

Replace all hardcoded date/time formatting in server.py and index.html with `Intl.DateTimeFormat` (JS) and `babel.dates` or `strftime` patterns (Python) that respect the user's browser locale.

## Key requirements

- JS: `new Intl.DateTimeFormat(navigator.language, {...}).format(date)` in commits feed
- Python: use `datetime.strftime` patterns consistent with ISO 8601
- no hardcoded `%-d` (Linux-only) format specifiers

## Expected file families

- JS changes in `web/dashboard/index.html` (≈20 lines); Python changes in `web/dashboard/server.py` (≈10 lines)
