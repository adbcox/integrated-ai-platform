## Description

Replace all `toFixed()` and manual percentage strings with `Intl.NumberFormat` so numbers render correctly for the user's locale (e.g., `1.234,56` vs `1,234.56`).

## Key requirements

- `function fmtNum(n, opts={})` wrapping `Intl.NumberFormat(navigator.language, opts).format(n)`
- `function fmtPct(n)` for percentage display
- applied to all numeric outputs in Overview and Analytics tabs
