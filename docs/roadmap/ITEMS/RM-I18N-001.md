## Description

Extract all user-visible strings from index.html into a JS `STRINGS` dictionary keyed by slug, then replace inline strings with `t('key')` calls, making future translation straightforward.

## Key requirements

- `const STRINGS = { en: { start_executor: 'Start Executor', ... } }` object
- `function t(key)` returns `STRINGS[currentLang][key] ?? key`
- all button labels, headings, and status messages extracted
