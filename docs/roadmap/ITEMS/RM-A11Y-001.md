## Description

Add `role=region`, `aria-label`, `role=list`, `role=listitem`, and descriptive `aria-label` attributes to all kanban columns and cards so screen readers can navigate the board meaningfully.

## Key requirements

- `role=region aria-label='Kanban board'` on container
- each column: `role=list aria-label='{Status} — {N} items'`
- each card: `role=listitem aria-label='{ID}: {title}'`

## Expected file families

- HTML attribute additions in `web/dashboard/index.html` kanban template (≈30 lines changed)
