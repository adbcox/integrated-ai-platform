## Description

Add a visually-hidden 'Skip to main content' anchor as the first focusable element on the page, becoming visible on focus, so keyboard users can bypass the header and tab navigation.

## Key requirements

- `<a href='#main-content' class='skip-link'>Skip to main content</a>` as first child of body
- CSS: hidden by default, `position:absolute; top:-100px`; visible on `:focus`
- `id='main-content'` added to main content area

- HTML + CSS in `web/dashboard/index.html` (≈20 lines added)
