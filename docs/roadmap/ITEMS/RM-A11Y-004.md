## Description

Add a `@media (forced-colors: active)` block that replaces custom color variables with `ButtonText`, `ButtonFace`, and `Highlight` system colors so the dashboard remains usable in Windows High Contrast mode.

## Key requirements

- `@media (forced-colors: active)` CSS block
- replaces all `var(--clr-*)` in interactive elements
- tested against WCAG 1.4.11 Non-text Contrast criterion
