## Description

Add a compact language selector (globe icon + dropdown) to the dashboard header that lets users switch between available UI languages (initially en, es, fr stubs), storing choice in localStorage.

## Key requirements

- `<select id='lang-select'>` with emoji flags and language names
- `switchLang(code)` sets `currentLang`, calls `applyStrings()`, saves to localStorage
- language loaded from localStorage on page init
