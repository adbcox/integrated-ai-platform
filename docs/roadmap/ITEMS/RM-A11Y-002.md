## Description

When a modal (training, shortcuts, item detail) opens, trap Tab/Shift-Tab focus inside the modal and restore focus to the triggering element when the modal closes.

## Key requirements

- `trapFocus(el)` and `releaseFocus()` JS utilities
- applied to all existing modal open/close paths
- Escape key closes modal and restores focus

## Expected file families
