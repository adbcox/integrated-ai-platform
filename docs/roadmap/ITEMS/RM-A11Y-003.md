## Description

Add an `aria-live=polite` region that announces executor state transitions (Started, Completed item X, Stopped) so screen reader users receive the same real-time feedback as sighted users.

## Key requirements

- `<div aria-live='polite' aria-atomic='true' class='sr-only'>` in HTML
- JS updates its text content on every `pollExec()` state change
- debounced to max 1 announcement per 3 s

## Expected file families

- HTML + JS in `web/dashboard/index.html` (≈25 lines added)
