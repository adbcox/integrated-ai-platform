# Feature Block Grouping

Feature block grouping exists to cluster roadmap items that should be planned and executed together because they share file families, system surfaces, validation steps, or architectural authority boundaries.

Initial seed groups:

- `FB-GOV-ROADMAP-AUTHORITY`
- `FB-DEV-AI-CONTINUITY`

Grouping must remain machine-readable in roadmap items and must not rely only on freeform narrative tags.
