# System Mission And Scope

## Mission
Advance local-first coding capability toward Codex 5.1 replacement and Version 15 with measurable gains in first-attempt quality and bounded complex-task completion.

## Scope for this wave
- Implement governed reuse wrappers for OpenHands, MarkItDown, MCP reference servers, and PR-Agent.
- Keep n8n in a bounded evaluation lane only.

## Out of scope for this wave
- Full Dify, LocalAI, RAGFlow, or n8n platform rollouts.
- Fine-tuning rollout changes.
- Parallel architecture creation.

## Success conditions for this wave
- Repo-visible executable surfaces exist.
- Validation commands are documented and runnable.
- Rollback/disable paths are explicit.
- Roadmap/governance truth surfaces are synchronized.
