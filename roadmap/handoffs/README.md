# Legacy Handoffs Subtree

This directory is part of the legacy top-level `roadmap/` tree.

## Canonical roadmap location

- `docs/roadmap/`
- `docs/roadmap/ROADMAP_INDEX.md`
- `docs/roadmap/EXECUTION_CONTEXT_SYNC_2026-04-21.md`

## How to read files here

Treat markdown in `roadmap/handoffs/` as legacy handoff and execution-context archive material unless a file explicitly states it has been migrated.

- New normalized roadmap maintenance does not belong here.
- Handoff artifacts here may still provide historical execution context.
- New issues, PRs, and execution packages should reference canonical roadmap IDs from `docs/roadmap/ROADMAP_INDEX.md`.
- If there is any conflict with `docs/roadmap/`, the canonical `docs/roadmap/` version wins.
