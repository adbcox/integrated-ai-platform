# Roadmap

This directory is the canonical planning and execution documentation root for the integrated-ai-platform project.

## Structure

- `roadmap_master.md` — primary roadmap and package progression summary
- `status/` — current phase and progress snapshots
- `planning/` — short planning-window prompts and planning artifacts
- `handoffs/` — execution-ready handoffs and large text artifacts
- `reports/` — execution reports, validation summaries, and milestone reports

## Operating rule

Chat remains the control surface for short planning prompts. Long handoffs, large code-adjacent specifications, and milestone documentation should be stored here so the repo is the durable source of truth.
